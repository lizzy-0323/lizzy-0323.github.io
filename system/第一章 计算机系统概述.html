# 计算机系统概述

# 操作系统的基本概念

计算机系统自上而下可以划分为：硬件、操作系统、应用程序和用户 操作系统的定义：是指控制和管理整个计算机系统的硬件与软件资源的程序集合，是计算机系统中最基本的系统软件。

# 操作系统的特征

  • 并发
  • 共享：分为互斥和同步
  • 虚拟化
  • 异步 并发和并行的区别：并发是指一段时间内，宏观上有多道程序运行，并行是指系统同时进行运算和操作。

# 操作系统的功能

  • CPU管理：包括进程同步、进程控制、进程通信、死锁处理、进程调度
  • 存储器管理：内存管理
  • 文件管理
  • 设备管理

# 操作系统的发展阶段

  • 人工操作阶段
  • 单道批处理与多道批处理阶段
  • 分时操作系统：主打同时性和用户的可交互性
  • 实时操作系统
  • 网络操作系统与分布式操作系统

# 中断和异常的概念

中断和异常的定义：中断是外中断，与程序无关。内中断是指地址越界，算术溢出等 中断处理过程：

  • 关中断
  • 保存断点
  • 中断服务程序寻址
  • 保存现场和屏蔽字
  • 开中断
  • 执行中断服务子程序
  • 关中断
  • 恢复现场和屏蔽字
  • 开中断，中断返回
