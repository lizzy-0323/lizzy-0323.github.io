# 计算机网络概述[[第一章 概论]]

# 计算机网络的组成

  • 从组成部分上看：分为软件、硬件、协议三大部分
  • 从工作方式上看： 计算机网络指边缘部分和核心部分
  • 从功能组成上看：分为通信子网和资源子网。

# 计算机网络的分类

  • 广域网：WAN
  • 域域网：MAN
  • 局域网：LAN 按交换技术分类：
  • 电路交换:建立真实的通路用于传送数据，优点是数据直接传输，时延小，缺点是线路的利用率低，没有差错控制
  • 报文交换：采用store and forward网络，优点是充分利用了线路容量，缺点是增加了资源开销
  • 分组交换：优点是易于管理，平均时延更小，所以优于报文交换

# 计算机网络性能指标

  • 带宽：最高传输速率
  • 时延：分为发送时延（分组长度/带宽），传播时延、处理时延、排队时延
  • 时延带宽积：传播时延X信道带宽
  • 往返时延：RTT，round trip time
  • 吞吐量：单位时间的吞吐量
  • 信道利用率：有数据通过的时间/总时间

# 计算机网络分层[[第一章 概论]]

# OSI模型与TCP/IP模型

# OSI

分为物理层，数据链路层，网络层，传输层，会话层，表示层，应用层。 低三层是通信子网，高三层是资源子网。 各层的单位：

  • 物理层：bit
  • 数据链路层：frame
  • 网络层：datagram
  • 传输层：segment
