# 通信基础[[第二章 物理层]]

# 基本概念

  • 码元：用一个固定时长的信号表示波形
  • 数据
  • 基带传输：0，1
  • 宽带传输：模拟信号

# 奈奎斯特定理与香农定理

# 奈奎斯特定理

用于计算理想状态下的信道传输速率

# 香农定理

信道的极限传输速率，不仅考虑带宽，还考虑信噪比S/N

# 编码与调制

编码：变为数字信号 调制：变为模拟信号

# 传输介质

# 有线介质

UTP-双绞线 STP-屏蔽双绞线 同轴电缆 光纤-利用光的全反射

# 无线介质

无线电波 微波、红外线和激光

# 物理层接口特性

  • 机械特性
  • 电气特性
  • 功能特性：电压的意义
  • 过程特性：事件的出现顺序

# 物理层设备

中继器：放大数字信号 集线器：HUB，是一个标准的共享式设备，只能在半双工情况下工作 这里注意一下，有集线器的情况下，带宽的计算方式
