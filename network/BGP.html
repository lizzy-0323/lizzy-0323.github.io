# BGP

Border Gateway Protocol 边界网关协议

# 定义

不同的自治系统之间交换路由信息的外部网关协议 目的：找到一条能够到达目标网络且比较好的路由

# 流程

每个AS的管理员选择一个路由器作为发言人，然后与另外一个AS的路由器建立TCP连接 刚运行时，需要交换整个BGP路由表，后续只有在有变化时再交换
